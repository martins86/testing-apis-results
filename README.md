### Comandos

## Iniciando o Git

```
git init
```

## Iniciando o package.json

```
npm init -y
```

## Instalando e iniciando o Jest

```
npm install -g jest
npm install --save-dev jest @types/jest
npm install --save ts-node ts-jest typescript cross-fetch
jest --init
```
